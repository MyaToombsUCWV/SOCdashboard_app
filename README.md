# Sentinel SOC Dashboard

## Project Overview
Sentinel SOC Dashboard is a Flutter-based prototype Security Operations Center dashboard designed to simulate monitoring security alerts, endpoint devices, reports, and analyst settings.

## Features
- Prototype authentication login
- Dashboard analytics
- Security alerts monitoring
- Device monitoring
- Reporting module
- Settings screen
- Local mock SOC data simulation
- Interactive visualizations

## Screens
1. Login Screen
2. Dashboard Screen
3. Alerts Screen
4. Devices Screen
5. Reports Screen
6. Settings Screen

## APIs / Packages Used
- Flutter Material API
- Flutter Navigation API
- fl_chart
- SharedPreferences
- Dart collections and data models

## Architecture
Project organized using:
- screens/
- models/
- data/

## Run Instructions
```bash
flutter pub get
flutter run
```

Build APK:

```bash
flutter build apk
```
